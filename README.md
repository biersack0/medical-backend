## Description
API for a medical center with google authentication and mongoDB, developed with [Nest](https://github.com/nestjs/nest) framework TypeScript.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start:dev
```

## Open the app
```bash
http://localhost:3005/
```
